# Tienda App 🛒

Sistema de gestión de inventario y ventas desarrollado en Python con una arquitectura MVC (Modelo-Vista-Controlador).

## 🚀 Características

- **Gestión de Productos:** Creación, listado, búsqueda y control de stock.
- **Categorías:** Organización de productos por categorías.
- **Clientes:** Registro y administración de base de datos de clientes.
- **Ventas:** Sistema de registro de ventas con detalles de múltiples productos.
- **Utilidades:** Validación de entradas de datos para asegurar la integridad de la información.

## 🛠️ Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/pedrodanielpereztorres-maker/tienda_app.git
   cd tienda_app
   ```

2. Crea un entorno virtual:
   ```bash
   python -m venv venv
   # En Windows:
   .\venv\Scripts\activate
   # En Linux/Mac:
   source venv/bin/activate
   ```

3. Instala las dependencias (si las hay):
   ```bash
   pip install -r requirements.txt
   ```

## 💻 Uso

Ejecuta la aplicación principal (asegúrate de tener un archivo de entrada como `main.py` que llame a las vistas):
```bash
python main.py
```
